## 📋 Sobre o Projeto

Projeto focado no estudo de **design responsivo**, com aplicação de técnicas modernas de CSS para criar layouts que se adaptam a diferentes tamanhos de tela: mobile, tablet e desktop.

---

## ✨ O que foi praticado

- 📐 **Flexbox** — alinhamento e distribuição de elementos
- 🔲 **CSS Grid** — layouts em grade
- 📱 **Media Queries** — adaptação para diferentes dispositivos
- 📏 Unidades relativas (`rem`, `em`, `%`, `vw`, `vh`)
- 🎨 Técnicas de layout moderno
